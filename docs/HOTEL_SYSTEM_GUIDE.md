## How It Works: The Simple Flow

### **Step 1: Guest Inquiry**
1. Guest visits your website
2. Fills out basic information (name, email, phone)
3. Selects booking purpose (Business/Leisure/Group)
4. Adds any special requests or preferences
5. System creates secure guest profile

### **Step 2: Room Booking Request**
1. Guest selects preferred room type
2. Chooses check-in and check-out dates
3. Adds special requests (late check-in, dietary needs, etc.)
4. Submits booking request (takes 1 minute)

### **Step 3: Your Admin Dashboard**
1. You see new booking requests immediately
2. Review guest details and special requests
3. Click "Confirm" to approve booking
4. System automatically sends SMS confirmation to guest

### **Step 4: Guest Receives Confirmation**
1. Guest gets SMS: "Your booking is confirmed. Check-in: [date] at 3 PM, Check-out: [date] at 11 AM. Room: [type]"
2. Guest can view booking details anytime
3. Easy to modify or cancel if needed

**Q: Do guests need to download an app?**

**Q: Can we still take phone bookings?**
